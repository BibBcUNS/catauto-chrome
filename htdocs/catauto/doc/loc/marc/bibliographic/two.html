This example can be identified as a record for a two-dimensional nonprojectable graphic by code k in Leader/06, and illustrates the kind of record used for an original or historical item. Noteworthy MARC data elements in this example include: field 100, subfield $e (Relator term); field 245, subfield $h (Medium); field 541 (Immediate Source of Acquisition Note); field 655 (Index Term Genre/Form); and second indicator value 7 in fields 650 and 655, used to cite the source of the term.

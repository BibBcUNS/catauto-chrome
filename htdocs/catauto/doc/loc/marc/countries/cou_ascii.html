MARC Country Codes

Code Sequence - ASCII Version

INTRODUCTION

This is an ASCII list of all valid and obsolete country codes assigned for use in MARC 21 records. It is intended to meet the need for a simple list to support MARC system development and online validation of MARC codes. This listing does not include any usage guidelines, or cross reference information. This information is available in the printed MARC code list and the full web version, available at: http://www.loc.gov/marc/countries/cou_home.html

Discontinued codes are identified by a hyphen preceding the code

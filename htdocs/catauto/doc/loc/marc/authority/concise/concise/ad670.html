670 - Source Data Found (R)

MARC 21 Authority - Concise
2007

Citation for a consulted source in which information is found about the 1XX heading in an established heading record, an established heading and subdivision record, a subdivision record or a reference record. May also include the information found in the source.

Indicators
First - Undefined
# - Undefined
Second - Undefined
# - Undefined
Subfield Codes
$a - Source citation (NR)
$b - Information found (NR)
$u - Uniform Resource Identifier (R)
$6 - Linkage (NR)
See description of this subfield in Appendix A: Control Subfields.
$8 - Field link and sequence number (R)
See description of this subfield in Appendix A: Control Subfields.
